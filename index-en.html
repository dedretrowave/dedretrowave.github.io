WHO AM I?

My name is Dzmitry, I am a web developer from Belarus. My main specialization is the creation of turnkey web applications and websites. I got the basis of my skills at a course on website development from the IT Academy "STEP". Then I went through a long way of self-education and work on various projects, the list of which will be below. At the moment, I am a high-tier specialist who is able to cope with many projects alone, and with almost any project when working in a good and cohesive team. My communication skills and focus on results perfectly help me in this. In addition, I speak English at the C1 level, which allows me to increase the range of my possibilities for implementation.

WHAT TECHNOLOGIES DO I WORK WITH?

List of technologies I specialize in

Front-End

JavaScript

JavaScript is my main programming language, and I use it on a regular basis. I have a lot of experience working with it and developing amazing interfaces and web services on it.

React

I consider React to be the best JavaScript library at the moment, as I really like its simplicity and well-supported ecosystem.

Bootstrap

Bootstrap is the #1 CSS library for me. I choose Bootstrap if the project conditions require it or for better code support.

jQuery

Despite the fact that jQuery is quite outdated for a modern Front-End, sometimes it is necessary in some projects. And fortunately, I have experience working with it.

Angular

Angular is a pretty convenient solution for large applications. I met him while working at a fintech startup, and I liked his architecture.

Back-End

NodeJS

Since JavaScript is my main language, I prefer to use NodeJS as a BackEnd platform for my applications. It is really fast and convenient to use

PHP

PHP is still the most convenient language for the back-end in the development of projects of any scale. I like it and I use it regularly

Opencart

Opencart is a great choice for an ecommerce site, and in my projects they are usually built on it.

wordpress

Wordpress is a CMS that gives a huge number of possibilities thanks to plugins, I use it if there is no specialized solution.
