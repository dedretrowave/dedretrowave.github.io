КТО Я И ЧТО Я УМЕЮ?

Меня зовут Дмитрий, я веб разработчик из Беларуси. Основная моя специализация - создание веб приложений и сайтов под ключ. Основу своих навыков я получил на курсе по разработке Веб Сайтов от IT Академии "ШАГ". Далее я прошёл долгий путь самообразования и работы на различных проектах, список которых будет ниже. На данный момент я являюсь специалистом среднего уровня, способный справиться с множеством проектов в одиночку, и практически с любым проектом при работе в хорошей и сплочённой команде. Мои коммуникативные навыки и нацеленность на результат отлично помогают мне в этом. Ко всему прочему я владею английским языком на уровне C1, что позволяет мне увеличивать диапазон своих возможностей для реализации.

С КАКИМИ ТЕХНОЛОГИЯМИ Я РАБОТАЮ?

Список технологий на которых я специализируюсь

Front-End

JavaScript

JavaScript - мой основной язык программирования, и я использую его на постоянной основе. У меня большой опыт работы с ним и разработки потрясающих интерфейсов и веб сервисов на нём.

React

Я считаю React лучшей библиотекой JavaScript на данный момент, так как мне очень нравится ее простота и отлично поддерживаемая экосистема.

Bootstrap

Bootstrap является для меня библиотекой CSS №1. Я выбираю Bootstrap, если этого требуют условия проекта или для лучшей поддержки кода.

jQuery

Несмотря на то, что jQuery довольно устарел для современного Front-End, иногда в некоторых проектах он необходим. И к счастью, у меня есть как опыт работы с ним, так и опыт выполнения тех же задач на ванильном JS

Angular

Angular - довольно удобное решение для больших приложений. Я познакомился с ним во время работы в финтех-стартапе, и мне понравилась его архитектура.

Back-End

NodeJS

Поскольку JavaScript - мой основной язык, я предпочитаю использовать NodeJS в качестве платформы Back-End для своих приложений. Она действительно быстрая и удобная в работе

PHP

PHP по-прежнему остается самым удобным языком для back-end в вопросе разработки проектов любого масштаба. Мне он нравится и я регулярно его использую

Opencart

Opencart - это отличный выбор для сайта электронной коммерции, и в моих проектах они обычно построены на нем.

wordpress

Wordpress - Эта CMS даёт огромное количество возможностей благодаря плагинам, я использую её в случае если нет какого-либо специализированного решения.
